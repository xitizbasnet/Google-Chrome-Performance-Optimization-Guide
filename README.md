# Google Chrome Performance Optimization Guide

> **Purpose:** Improve Google Chrome performance by optimizing built-in settings that reduce RAM and CPU usage.
>


---

# Overview

Have you ever noticed that your computer becomes significantly slower while using Google Chrome?

In many cases, even after closing Chrome, your PC may continue to feel slow and sluggish.

Although Google Chrome is one of the best web browsers available today, several default settings can quietly consume large amounts of **RAM** and **CPU** resources. Some background processes may continue running even after the browser has been closed.

The good news is that a few simple configuration changes can make Chrome:

* 🚀 Faster
* 💾 More memory efficient
* ⚡ Less CPU intensive
* 🖥️ More responsive

This guide explains the recommended settings, what each setting does, and when you should enable or disable them.

---

# Optimization 1: Stop Chrome from Running in the Background

## 📍 Navigation

Open Google Chrome and enter the following in the address bar:

```text
chrome://settings/system
```

Press **Enter**.

This opens the **System** settings page.

---

## Configuration

Locate the following option:

> **Continue running background apps when Google Chrome is closed**

### Recommended Setting

✅ **Disable (Turn Off)**

---

## What This Setting Does

When this option is enabled:

* Chrome does **not** completely shut down after closing the browser.
* Extensions continue running.
* Web applications remain active.
* Background processes continue consuming RAM and CPU resources.

As a result:

* Your computer may remain slow.
* Chrome may continue using system resources throughout the day.
* Performance may decrease without any visible Chrome window being open.

---

## Benefits

Disabling this setting:

* Frees RAM
* Reduces CPU usage
* Stops unnecessary background processes
* Improves overall system responsiveness

> 💡 For many users, this single change provides one of the biggest performance improvements.

---

# Optimization 2: Enable Memory Saver

## 📍 Navigation

Open Chrome and enter:

```text
chrome://settings
```

Press **Enter**.

From the left navigation menu:

**Performance**

Locate:

> **Memory Saver**

---

## Recommended Setting

✅ **Enable Memory Saver**

---

## What Memory Saver Does

Every open browser tab consumes memory.

When dozens of tabs remain open, Chrome can use a significant amount of RAM—even if many of those tabs are inactive.

Memory Saver automatically:

* Places inactive tabs into sleep mode after they have not been used for some time.
* Frees system memory.
* Reloads the tab automatically when it is selected again.

This provides a noticeable performance improvement, particularly for users who keep many tabs open.

---

# Memory Saver Modes

After enabling Memory Saver, three operating modes are available.

---

## 1. Moderate

### Description

Tabs remain active for a longer period before entering sleep mode.

### Best For

* Users who frequently switch between tabs
* Systems with sufficient RAM

---

## 2. Balanced (Recommended)

### Description

Provides a balance between:

* Keeping tabs readily available
* Freeing memory when required

### Best For

* Most users
* Everyday browsing

> ✅ This is Chrome's recommended setting.

---

## 3. Maximum

### Description

Inactive tabs are put to sleep much sooner.

### Benefits

* Maximum memory savings
* Lower RAM usage
* Better performance on low-memory systems

### Best For

* Computers with limited RAM
* Systems that become slow with many open tabs

> 💡 If your PC struggles when multiple tabs are open, try **Maximum** mode for improved responsiveness.

# Optimization 3: Clean Up Unused Extensions

## 📍 Navigation

Open a new tab and enter:

```text
chrome://extensions
```

Press **Enter**.

This page displays every extension currently installed in Chrome.

---

## Review Installed Extensions

Take a moment to review your installed extensions.

Ask yourself:

> **How many of these extensions do you actually use?**

Many users install extensions for a temporary purpose and then forget about them.

---

## Why This Matters

Most extensions do **not** remain idle.

Many continue running whenever Chrome starts.

Depending on the extension, they may consume:

* RAM
* CPU resources
* Browser processing time

Common examples include:

* Ad blockers
* Password managers
* Shopping assistants
* Screenshot tools
* AI assistants
* Productivity extensions

While each extension may have only a small impact individually, multiple active extensions can noticeably reduce browser performance.

---

## Recommended Action

Review your extensions regularly.

### Remove

Delete extensions that:

* Have not been used for weeks or months
* Are no longer needed

### Disable

If you may need an extension later, disable it instead of removing it completely.

---

## Benefits

Cleaning up unused extensions can:

* Improve Chrome startup time
* Reduce memory consumption
* Lower CPU usage
* Improve browsing performance

> ⚠️ Many users overlook this simple maintenance task, especially if Chrome has been used for a year or longer without reviewing installed extensions.

# Best Practices

* ✅ Disable Chrome background applications unless specifically required.
* ✅ Enable Memory Saver.
* ✅ Use **Balanced** mode for most computers.
* ✅ Use **Maximum** mode on low-memory systems.
* ✅ Review installed extensions regularly.
* ✅ Remove or disable extensions that are no longer needed.

# Conclusion

Google Chrome is an excellent browser, but its default configuration may consume more system resources than necessary.

By making the three configuration changes described in this guide, you can significantly improve browser performance and reduce unnecessary RAM and CPU usage without sacrificing essential functionality.
